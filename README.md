AfriScope is a data analytics project built in R that explores
economic and currency trends in East Africa using public data.

## Objectives
- Analyze inflation, GDP, and exchange rates
- Identify macroeconomic patterns
- Communicate insights through visualizations and dashboards

## Tools Used
- R, RStudio
- tidyverse, ggplot2, Shiny
- Git & GitHub

## Key Insights
- Inflation spikes correlate with currency pressure
- GDP growth shows lagged effects on price stability
